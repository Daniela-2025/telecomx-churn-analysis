# 📊 TelecomX - Análisis de evasión de clientes

## 📌 Sobre este proyecto

Hola 👋

Este proyecto forma parte de un challenge de análisis de datos donde tuve que investigar el problema de **evasión de clientes (churn)** en una empresa ficticia llamada TelecomX.

La empresa estaba teniendo muchos clientes que cancelaban su servicio y la idea del análisis era tratar de entender **qué cosas podrían estar influyendo en que los clientes se vayan**.

Para eso trabajé con un dataset que incluye información como:

* servicios contratados
* tipo de contrato
* cargos mensuales
* tiempo que el cliente lleva en la empresa
* y otras características del cliente

El objetivo fue **limpiar los datos, explorarlos y tratar de encontrar algunos patrones** que puedan ayudar a entender mejor este problema.

No es un análisis perfecto, pero fue un buen ejercicio para practicar todo el proceso de análisis de datos.

El notebook sigue más o menos las etapas típicas de un proceso de análisis de datos:

## 📊 Algunas cosas interesantes que se observaron

Durante el análisis aparecieron algunos patrones que llamaron la atención:

* Los **contratos mensuales** parecen tener más churn que los contratos más largos.
* Los clientes que **tienen menos servicios contratados** tienden a cancelar más.
* En algunos casos los **cargos mensuales más altos** también parecen estar relacionados con mayor evasión.

Esto no significa que sea la causa exacta del problema, pero sí da pistas que podrían investigarse más a fondo.
